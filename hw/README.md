# Задания

### 1. Напишите функцию <span style="color:green">min(a, b)</span>, которая возвращает меньшее из двух чисел.
- Используйте <span style="color:green">if..else</span>.
- Попробуйте записать с помощью тернарного оператора.

### 2. Создайте функцию <span style="color:green">power(x, n)</span>, которая возвращает x в степени n.

### 3. Напишите функцию <span style="color:green">factorial(n)</span>, которая возвращает факториал числа n.
- Реализуйте с <span style="color:green">while</span>.

### 4. Создайте функцию <span style="color:green">reverseString(str)</span>, которая принимает строку и возвращает ее в обратном порядке.
- Используйте <span style="color:green">for</span>.

### 5. Создайте функцию <span style="color:green">isPalindrome(str)</span>, которая проверяет, является ли строка палиндромом (читается одинаково слева направо и справа налево).
- Используйте <span style="color:green">if..else</span> внутри цикла.

### 6. Создайте функцию <span style="color:green">findMax(arr)</span>, которая принимает строку из чисел и возвращает максимальное число.
- Используйте <span style="color:green">for</span>.

### 7. Напишите функцию <span style="color:green">fizzBuzz(n)</span>, которая выводит числа от <span style="color:green">1</span> до <span style="color:green">n</span>, но вместо чисел, кратных <span style="color:green">3</span>, выводит <span style="color:green">"Fizz"</span>, вместо чисел, кратных <span style="color:green">5</span> — <span style="color:green">"Buzz"</span>, а вместо чисел, кратных <span style="color:green">3</span> и <span style="color:green">5</span> — <span style="color:green">"FizzBuzz"</span>.
-Используйте <span style="color:green">for</span> и <span style="color:green">if..else</span>.

### 8. Создайте функцию <span style="color:green">generateRandomArray(size, min, max)</span>, которая возвращает строку из <span style="color:green">size</span> случайных чисел в диапазоне от <span style="color:green">min</span> до <span style="color:green">max</span>.

### 9. Напишите функцию <span style="color:green">applyDiscount(price, discountFunction)</span>, где <span style="color:green">discountFunction</span> — это колбэк, вычисляющий скидку.
- Проверьте, передав функции <span style="color:green">noDiscount</span> (возвращает ту же цену) и <span style="color:green">halfPrice</span> (уменьшает цену в 2 раза).

### 10. Напишите функцию <span style="color:green">countVowels(str)</span>, которая считает количество гласных (<span style="color:green">a, e, i, o, u</span>) в строке.
- Используйте <span style="color:green">for</span> и <span style="color:green">if</span>.

### 11. Создайте функцию <span style="color:green">getDayName(dayNumber)</span>, которая принимает число от <span style="color:green">1</span> до <span style="color:green">7</span> и возвращает название дня недели.
- Используйте <span style="color:green">switch</span>.

### 12. Создайте функцию <span style="color:green">isPrime(n)</span>, которая проверяет, является ли число простым.

### 13. Создайте функцию <span style="color:green">countOccurrences(srt, target)</span>, которая считает количество вхождений <span style="color:green">target</span> в строку <span style="color:green">str</span>.

### 14. Создайте функцию <span style="color:green">capitalizeWords(str)</span>, которая делает заглавной первую букву каждого слова в строке.

### 15. Напишите функцию <span style="color:green">generatePassword(length)</span>, которая генерирует случайный пароль из букв и цифр заданной длины.

### 16. Создайте игру "Угадай число".
- Функция <span style="color:green">guessNumberGame()</span> загадывает случайное число от <span style="color:green">1</span> до <span style="color:green">100</span> и предлагает пользователю угадывать его, давая подсказки <span style="color:green">"Больше"</span> / <span style="color:green">"Меньше"</span>, пока число не будет угадано.
- Используйте <span style="color:green">while</span>.

### 17. Создайте функцию <span style="color:green">calculate(a, b, operator)</span>, которая выполняет арифметические операции <span style="color:green">"+"</span>, <span style="color:green">"-"</span>, <span style="color:green">"*"</span>, <span style="color:green">"/"</span>, принимая их в качестве третьего параметра.
- Используйте <span style="color:green">switch</span>.
